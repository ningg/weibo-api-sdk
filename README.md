# weibo-api-sdk
weibo api SDK(Java)


## 概要

简要说明:

* 当前工程为 weibo api SDK(Java 版本)。
* 基于 [weibo4j-oauth2-beta3.1.1](https://github.com/sunxiaowei2014/weibo4j-oauth2-beta3.1.1) 重新整理改进。

当前工程覆盖的 weibo api:

* ALL From [微博 API 官网](http://open.weibo.com/wiki/%E5%BE%AE%E5%8D%9AAPI)
